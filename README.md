## Instructions

1. In the `backend` directory, build the project and start the Spring Boot server

```
mvn clean install
mvn spring-boot:run
```

2. In the `frontend` directory, install the dependencies, and start the web server

```
npm install
npx http-server
```

3. In the `frontend` directory, run the tests

```
node test.js
```
